📘 Памятка и правила

🏆 Поощрения

  • Тренировки: 10–20 мин → 50₽, 20–40 мин → 100₽, 40–60 мин → 150₽, >60 мин → 200₽
  • Питание: Соблюдение калорийности (весь день) → 100₽, норма воды → 50₽
  • Контроль: Взвешивание утром → 10₽, замер объёмов (раз в неделю) → 30₽
  • Результаты (по умным весам): -1 кг → 1000₽, -1% жира → 200₽, сохранение мышц → 150₽/неделя
  • Бонусы: Неделя без срывов → 300₽, промежуточная цель (ваша) → 1000₽

⚠️ Штрафы

  • Газировка: до 0.25л → 150₽, больше → 500₽
  • Сладости: малый кусок торта/пирожное → 150₽, большой → 500₽
  • Конфеты/печенье: до 50г → 100₽, больше → 300₽
  • Фастфуд: фри/наггетсы → 300₽, бургер/шаурма → 400₽
  • Чипсы/сухарики → 200₽, булочка/круассан → 100₽
  • Пропуск тренировки без уважительной причины → 100₽

📌 Автоматические бонусы

При внесении новых измерений веса и % жира система сравнивает с предыдущим значением. Если зафиксировано снижение на 1 кг или 1% жира — автоматически добавляется соответствующий бонус (1000₽ за кг, 200₽ за % жира). Бонусы могут начисляться несколько раз, если потеряно несколько кг или % за один раз.

💡 Советы

• Старайтесь записывать операции сразу после события.
• Используйте ручной ввод для нестандартных ситуаций.
• Еженедельно обсуждайте прогресс — это усиливает мотивацию.
• Баланс накапливается, можно тратить на приятные подарки или совместный досуг.

🌐 Сохранение данных

Все данные хранятся в вашем браузере (localStorage). При очистке кэша они могут исчезнуть. Для работы с нескольких устройств необходима серверная версия (например, с Firebase).
